Zero-Shot Real-World Robot Control via a Synthetic-Data Language–Action Model

Overview of SynthLA. The vision module converts scene images into structured textual descriptions, which are combined with the robot state and user instructions. SynthLA then generates the next low-level robot action.

Qualitative Experiments

We evaluate SynthLA in real-world scenarios under two conditions. In the static setting, the scene remains unchanged during inference. In the dynamic setting, objects may be displaced or new elements introduced during execution. For both conditions, we consider two data regimes: in-distribution (ID), where all objects were present in the fine-tuning dataset, and out-of-distribution (OOD), where the model encounters novel objects.

The experimental evaluation addresses the following research questions:

  1. How does SynthLA compare with a state-of-the-art (SOTA) vision–language–action (VLA) model across benchmark tasks? In particular, how does performance differ between ID and OOD conditions?
  2. How does performance scale with SynthLA's parameter size?
  3. How does SynthLA’s performance on a specific task scale with the number of tasks it has been trained on?
